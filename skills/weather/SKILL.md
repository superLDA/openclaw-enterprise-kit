# Weather Skill

基于 Open-Meteo 免费 API 的天气查询服务，无需任何 API Key。

## 功能

- 🌡️ 当前温度、体感温度
- 🌧️ 降水概率
- 💨 风速风向
- 🌅 日出日落时间
- 📅 7 天天气预报

## 用法

```
/weather 北京
/weather London
/weather Tokyo, 明天
```

## 示例

```
用户: /weather 上海
助手: 🌤️ 上海今日天气
温度: 22°C (体感 20°C)
湿度: 65%
风速: 3级 东南风
日出: 05:12 日落: 18:34
```
